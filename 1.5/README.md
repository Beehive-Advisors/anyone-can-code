## Recording order

1. Record the macOS terminal screencast (`1.5-script-macos.md`) natively on your Mac.
2. Record the Linux / WSL terminal screencast (`1.5-script-linux.md`) from a kubectl pod on the k0s cluster (preferred), a Docker Ubuntu container, WSL2, or a Linux VM.
